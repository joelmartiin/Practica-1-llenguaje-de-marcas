## Portafolios Web
## Descripción
Este es un proyecto de un portafolio web creado como primera práctica en la asignatura de Lenguaje de Marcas.
En él se muestra información personal sobre mí, incluyendo proyectos realizados, estudios cursados y habilidades tanto técnicas como sociales.

## Contexto
El objetivo de este proyecto es aplicar los conocimientos básicos de HTML y CSS aprendidos en clase, desarrollando una página web estructurada 
y funcional que sirva como presentación personal. Además, busca mejorar la organización del contenido y la navegación dentro de una web.
